# Explicacion 05 - Ejercicio 05

## En que consiste

El programa busca un numero ingresado por teclado y cuenta cuantas veces aparece en una lista doblemente enlazada.

## Estructuras usadas

- `Nodo`: guarda un entero y sus enlaces dobles.
- `ListaDoble`: permite insertar al final, mostrar la lista y contar coincidencias.
- `Scanner`: lee el valor que desea buscar el usuario.

## Idea clave

Se recorre la lista completa. Cada vez que el dato del nodo coincide con el dato buscado, se aumenta un contador.
