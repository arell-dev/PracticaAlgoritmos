# Explicacion 10 - Ejercicio 10

## En que consiste

El programa maneja un arreglo de postres. Cada postre tiene una lista enlazada simple de ingredientes.

## Estructuras usadas

- `Postre`: guarda el nombre y su lista de ingredientes.
- `SistemaPostres`: usa un arreglo `Postre[]` para almacenar los postres.
- `NodoIngrediente`: guarda el nombre de un ingrediente y el enlace al siguiente.
- `ListaIngredientes`: permite agregar, mostrar y eliminar ingredientes.

## Idea clave

La estructura principal es un arreglo de postres, y dentro de cada posicion hay una lista de ingredientes. Es una combinacion de arreglo + lista enlazada.
