# Explicacion 03 - Ejercicio 03

## En que consiste

El programa compara dos listas doblemente enlazadas para saber si tienen el mismo tamanio y el mismo contenido.

## Estructuras usadas

- `Nodo`: guarda un entero y sus enlaces `anterior` y `siguiente`.
- `ListaDoble`: guarda `cabeza`, `cola` y `tamanio`.

## Idea clave

Primero se compara el tamanio. Si el tamanio es igual, se recorren ambas listas al mismo tiempo para comparar dato por dato en la misma posicion.
