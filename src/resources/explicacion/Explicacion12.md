# Explicacion 12 - Ejercicio 12

## En que consiste

El programa representa una feria del libro con casetas, libros vendidos y firmas de autores.

## Estructuras usadas

- `FeriaLibro`: contiene listas de casetas y firmas.
- `Caseta`: contiene una lista de libros.
- `Libro`: guarda titulo y cantidad vendida.
- `Firma`: guarda autor, libro, hora y caseta.
- Listas enlazadas simples para casetas, libros y firmas.
- Arreglo auxiliar para calcular el top de libros vendidos.

## Idea clave

Las consultas se resuelven recorriendo listas: buscar libros por caseta, buscar casetas por libro, mostrar firmas por autor y ordenar ventas para obtener el top 10.
